Do you want to apply for a financial order?

If you disagree about dividing your money and property then you can ask the court to decide for you. This is known as applying for a financial order. The court needs to know now, if you want to apply. The application itself is done separately using another form, and costs an additional £255.

If you select yes:

  • you do not have to proceed with the application for a financial order.
  • you can proceed with the application for a financial order at any time, so long as your husband or wife is still alive.

If you select no:

  • you can still apply for a financial order in the future.
  • you’ll only be able to apply until you remarry or form a new civil partnership (this does not apply to pension sharing or pension compensation orders, which can be applied at any time)
