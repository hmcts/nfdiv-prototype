You need to apply for help with your divorce fees

You need to apply for help with your fees (opens in a new tab) before you continue with this divorce application.

Enter the court form number ‘D8’ when asked. This will be one of the first questions when you apply for help with your fees (opens in a new tab).

After you have applied then you will receive a Help With Fees reference number. You should enter the reference number when you return to this divorce application.

If you have a Help With Fees reference number then you can enter it here.
